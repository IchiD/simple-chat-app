## 実装されたファイル

### 新規作成ファイル

#### 1. `backend/app/Events/NewMessageEvent.php`
```php
<?php

namespace App\Events;

use App\Models\Message;
use Illuminate\Broadcasting\PrivateChannel;
use Illuminate\Contracts\Broadcasting\ShouldBroadcast;

class NewMessageEvent implements ShouldBroadcast
{
    public function broadcastOn(): array
    {
        return [
            new PrivateChannel('conversation.' . $this->message->conversation_id),
        ];
    }

    public function broadcastAs(): string
    {
        return 'new-message';
    }
    
    // メッセージデータの詳細なブロードキャスト
}
```

**特徴:**
- プライベートチャンネルでセキュアな配信
- 送信者情報を含む完全なメッセージデータ
- 管理者メッセージにも対応

#### 2. `frontend/plugins/echo.client.ts`
```typescript
import Echo from 'laravel-echo'
import Pusher from 'pusher-js'

export default defineNuxtPlugin(() => {
  // Laravel Echo設定
  const echoConfig = {
    broadcaster: 'reverb',
    key: config.public.reverbAppKey || 'app-key',
    wsHost: config.public.reverbHost || 'localhost',
    wsPort: config.public.reverbPort || 8080,
    // ... 認証ヘッダー設定

  window.Echo = echo
})
```

**特徴:**

## 残りの作業（環境セットアップ）

実装は完了していますが、以下の環境設定が必要です：

### 1. Laravel アプリケーションキー生成
```bash
cd backend
php artisan key:generate
```

### 2. Laravel Reverbサーバー起動
```bash
php artisan reverb:start
```

### 3. フロントエンド環境変数設定
`frontend/.env` ファイルに以下を追加：
```env
REVERB_APP_KEY=app-key
REVERB_HOST=localhost
REVERB_PORT=8080
REVERB_SCHEME=http
```

### 4. 開発サーバー起動
**バックエンド:**
```bash
cd backend
php artisan serve
```

**フロントエンド:**
```bash
cd frontend
npm run dev
```

**Reverbサーバー:**
```bash
cd backend
php artisan reverb:start
```
