## 技術仕様

### フロントエンド技術スタック

- **フレームワーク**: Nuxt 3 + Vue 3 Composition API
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **状態管理**: Pinia
- **HTTP 通信**: useApi composable

### バックエンド技術スタック

- **フレームワーク**: Laravel
- **決済処理**: Stripe API
- **認証**: Laravel Sanctum
- **データベース**: MySQL
